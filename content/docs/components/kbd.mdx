## About

The `Kbd` component is used to display keyboard keys and shortcuts in your UI. It helps users understand keyboard shortcuts and input methods.

## Usage

```tsx
import { Kbd, KbdGroup } from "@soar-design/soar-react-components";

<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>⇧</Kbd>
  <Kbd>⌥</Kbd>
  <Kbd>⌃</Kbd>
</KbdGroup>;
```
